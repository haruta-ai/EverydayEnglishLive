# Everyday English Live v0.6

外部APIを一切使用しない、日本アニメ調PWAです。

## 主な機能
- Mike / Sophie切り替え
- 太めスクエア眼鏡
- Sophieは髪を下ろしたスタイル
- まばたき、口パク、うなずき、手振り、共感表情
- 会話内容に応じた分岐
- 複数トピック
- 選択回答ごとに異なる次の質問
- 簡易キーワード理解による自由入力対応
- ブラウザ内記憶
- キャラクターごとの相づち
- GitHub Pages / iPhone対応
- APIなし

## 制限
AIではありません。自由入力はキーワード分類による簡易対応です。
