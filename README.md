# Everyday English Live v0.2.1

iPhone Safari / GitHub Pagesで試せる英会話PWAテスト版です。

## テスト機能
- Mike / Sophie切り替え
- 太めスクエア眼鏡
- Sophieは髪を下ろしたスタイル
- 英語音声読み上げ
- Safari対応範囲での音声入力
- 回答例と文字入力
- ホーム画面追加
- オフラインキャッシュ

## GitHub Pages
リポジトリ直下へ全ファイルをアップロードし、Settings > Pages から main / root を公開してください。
