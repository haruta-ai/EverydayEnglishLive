# Everyday English Live v0.4

外部APIを一切使わないPWAテスト版です。

## v0.4の変更
- 写真切り替え方式を廃止
- SVGパーツをリアルタイムに動かすアバター方式
- 目、口、首、肩、腕、手を個別にアニメーション
- まばたき、口パク、うなずき、手振り、説明ジェスチャー
- Mike / Sophie切り替え
- 質問ごとに対応した回答例を表示
- 会話の順番を状態管理
- iPhone / Mac対応PWA
- 外部APIなし

## 制限
実写動画ではなく、ブラウザ内SVGアニメーションです。
完全な自由会話や人間品質の音声には外部AIが必要ですが、この版では使用していません。
